## ğŸ’¡ TIPS & BEST PRACTICES

### Untuk Admin:

1. **Saat Tambah Petugas:**
   - Gunakan email yang valid & belum terdaftar
   - Password minimal 8 karakter
   - Pilih kecamatan sesuai wilayah kerja petugas

2. **Saat Edit Petugas:**
   - Kosongkan password jika tidak ingin mengubahnya
   - Email harus tetap unique
   - Pastikan data alamat benar

3. **Saat Hapus Petugas:**
   - Pastikan petugas tidak sedang handle data penting
   - Backup data jika perlu
   - Aksi tidak bisa dibatalkan!

### Untuk Developer:

1. **Maintenance:**
   ```bash
   # Clear cache setelah update
   php artisan cache:clear
   php artisan view:clear
   ```

2. **Debugging:**
   ```bash
   # Check database
   php artisan tinker
   >>> User::where('role', 'petugas')->count()
   ```

3. **Monitoring:**
   ```sql
   -- Query petugas per kecamatan
   SELECT alamat_kecamatan, COUNT(*) as total
   FROM users
   WHERE role = 'petugas'
   GROUP BY alamat_kecamatan;
   ```
